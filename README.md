An unofficial csrf protection for express.js.<br>
You use both __Esm__ and __Cjs__

# Install

```bash
  npm i express-simple-csrf
  yarn add express-simple-csrf
  pnpm i express-simple-csrf
  bun add express-simple-csrf
```
